# Caso de Uso – Cadastrar Filme

### Nome
Cadastrar Filme

### Ator
Funcionário/Adm

### Objetivo
Permitir que o funcionário cadastre novos filmes no sistema.

### Fluxo

1. O funcionário abre a tela de cadastro de filmes.
2. O sistema mostra o formulário.
3. O funcionário preenche os dados do filme.
4. O sistema verifica se as informações estão corretas.
5. O sistema salva o filme no banco de dados.
6. O sistema mostra uma mensagem de sucesso.
